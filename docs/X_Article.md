# Solana Agent Economy Hackathon — AgentMart (Demo)

Demo and technical writeup for the Agent-driven market making demo.

## What this is
- A demo agent that provides a passive market-making skill for SOL/USDC.
- Uses Bitget Wallet Skill for security checks and quote data.

## How to run (devnet / local)
1. Clone repo
2. Run demo script: bash mm/scripts/run_demo.sh
3. Or using Docker: docker compose up --build
